# Random Jokes

Random Jokes is a PHP library for generating random jokes

## Installation

Use the package manager [composer](https://getcomposer.org/download/) to install Random Jokes.

```bash
composer require gopibabus/random-jokes
```

## Usage

```php
use Gopibabu\Jokes\JokeFactory;

$randomJoke = new JokeFactory();
$randomJoke->generateJoke();
```
